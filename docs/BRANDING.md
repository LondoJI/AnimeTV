# AnimeTV Branding

This folder contains GitHub-ready branding assets for AnimeTV and JSolanoDev.

## Files

- `docs/branding/animetv-banner.svg` - README banner.
- `docs/branding/jsolanodev-avatar.svg` - vector avatar source.
- `docs/branding/jsolanodev-avatar.png` - upload-ready GitHub profile picture.

## Avatar Upload

1. Open GitHub.
2. Go to **Settings** > **Public profile**.
3. Click your profile picture.
4. Upload `docs/branding/jsolanodev-avatar.png`.
5. Save changes.

## Profile README

Use `github-profile/README.md` inside a repository named `JSolanoDev/JSolanoDev` to activate your GitHub profile README.
